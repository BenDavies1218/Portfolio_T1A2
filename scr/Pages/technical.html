SCSS (Super Cascading Style Sheets)

See how I write simple yet power SCSS to style and animate my anchor tags

    body {
        a {
            width: 120px;
            padding: 10px;
            text-align: center;
            text-decoration: none;
            font-family:'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
            font-weight: 900;
            color: black;
            background-color: rgb(104, 104, 255);
            border: 1px solid black;
            box-shadow: 5px 5px 0px 1px rgba(242, 235, 110, 0.7), 5px 5px 0 2px black;
            opacity: 0.8;
            transition: 0.1s;

            &:hover {
                opacity: 1;
                transform: translate(10px, 10px);
                box-shadow: none;
            }
        }
    }
